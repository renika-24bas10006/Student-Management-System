# Student Management System

## Overview
The following project is a Java-based console application for managing the following in an
educational institutions:
- Students
- Courses
- Grades

This system provides efficient data management with persistent storage capabilities.

## Features

### Student Management
- Add new students with complete details
- View all registered students
- Find students by ID
- Delete student records
- Input validation and error handling

### Course Management
- Create and manage course catalog
- Add course information (ID, name, credits, instructor)
- View all available courses
- Search courses by ID
- Delete course records

### Grade Management
- Assign grades to students for specific courses
- Automatic grade calculation (A, B, C, D, F)
- View all grade records
- Filter grades by student or course
- Grade validation and duplicate prevention
