# Student Management System - Project Statement

## Problem Statement
It is a major problem in the educational sector to efficiently manage and track
the student's records along with maintaining the course information and academic performance.
Manual record keeping becomes tedious and error-prone. This application aims to create a 
systematic storing and tracking space for Student's records, course records and grade records.


## Scope of the Project
- Comprehensive student information management
- Course catalog creation and maintenance
- Academic performance tracking through grade management
- Data persistence across application sessions
- User-friendly console interface for easy operation

## Target Users
- School and College Administrators
- Academic Coordinators
- Faculty Members
- Educational Institution Staff

## High-Level Features
1. **Student Registration System** - Complete student lifecycle management
2. **Course Management System** - Academic course organization and tracking
3. **Grade Management System** - Academic performance recording and reporting
4. **Data Persistence Layer** - Automatic data save and restore functionality
5. **Reporting Module** - Statistical overview and data analysis
